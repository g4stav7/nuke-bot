* Python 3.6 - 3.8
***
# Features
 - [x] Delete Channels
 - [x] Ban All Members
 - [x] Delete Roles
 - [x] Create Channel
 - [ ] Create Roles

# Installation:
## Android(Termux):
```console
apt update && apt upgrade

pkg install python git

git clone https://github.com/g4stav7/nuke-on.git

cd nuke-on

python -m pip install -r requirements.txt

python main.py
```
**[Subsequent launches]**
```console
cd nuke-on

python main.py
```
## Linux / Windows
```console
git clone https://github.com/g4stav7/nuke-on.git

cd nuke-on

python -m pip install -r requirements.txt

python3 main.py
```

***
# Running
It's that simple! Enter the command `python main.py` or `python3 main.py` and the nuke bot interface will launch.
