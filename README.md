# Simple nuke-on
* Python 3.6 - 3.8
***
# Features
 - [x] Delete Channels
 - [x] Ban All Members
 - [x] Delete Roles
 - [x] Create Channels
 - [ ] Create Roles

# Installation:
## Android(Termux):
```console
apt update && apt upgrade

pkg install python git

git clone 

cd nuke-on

python -m pip install -r requirements.txt

python nuke-on.py
```
**[Subsequent launches]**
```console
cd Discord-Nuke-Bot

python nuke-on.py
```
## Linux / Windows
```console
git clone 

cd nuke-on

python -m pip install -r requirements.txt

python3 nuke-on.py
```

***
# Running
It's that simple! Enter the command `python nuke-on.py` or `python3 nuke-on.py` and the nuke bot interface will launch.
